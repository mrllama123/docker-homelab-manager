> a simple web app to manage homelab docker compose services

## dev setup

### requirements

- [poetry](https://python-poetry.org/docs/#installation)
- [docker](https://docs.docker.com/get-docker/)
- python >=3.11

### setup

```bash
# install dependencies
poetry install
```

### running dev mode

```bash
# run the app
poe dev
```
